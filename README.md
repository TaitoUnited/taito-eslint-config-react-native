# Taito ESlint config for React Native

## Installation

```sh
npm install -D git+https://github.com/TaitoUnited/taito-eslint-config-react-native.git
```

## Usage

In `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@taito/react-native"
  }
}
```

Or in `.eslintrc` / `.eslintrc.json` / `.eslintrc.js`:

```json
{
  "extends": "@taito/react-native"
}
```
